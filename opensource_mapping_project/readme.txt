readme

Open source mapping project. This project pulls data from an online repository, performs data munging,
plots the data on a geographical map and does interpretation of the data.


1) acquire_data.sh
	shell script to acquire data, do data munging and call MATLAB script for plotting map

2) test_map_function.m
	MATLAB script for plotting map

3) test_map_function_generic.m
	MATLAB script for plotting map (generic function that will take any filename)

4) bigzone_map.jpg
	Generated map

5) interpretation.doc
	Interpretation and sanity check of map
